TREINANDO PROMISES

Imagine que estamos para partir numa viagem interplanetária e temos 3 naves à nossa disposição. Porém, as naves tem um
dispositivo que não permite a nossa partida com a bateria abaixo de 30% da capacidade máxima. 
Queremos dar a partida nas 3 naves e verificar qual com qual delas podemos sair. 

1 - Módulo para Naves com nome, carga atual e carga máxima.
2 - Módulo para dar partida nas naves, com método que checa o % de carga e retorna via Promise
3 - no index: instanciar as 2 classes e chamar o método de partida.
